ABSTRACT:


The Library Management System (LMS) is a browser‑based application designed to automate and simplify the management of books, members, borrowing, returning, due dates, and late fees. The system is developed using HTML, CSS, Bootstrap, and JavaScript and runs entirely on the client side without requiring installation, a database, or a backend server. It provides a professional interface for adding, editing, and deleting books and members, as well as borrowing and returning books with automatic due date and late fee calculation. The system also maintains a complete history log of all transactions. This project demonstrates how modern web technologies can be used to build an efficient, user‑friendly, and fully functional library management solution suitable for academic, small‑scale, and personal library environments.
