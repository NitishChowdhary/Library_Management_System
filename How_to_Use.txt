------------------------------------------------------------
HOW TO USE THE LIBRARY MANAGEMENT SYSTEM

------------------------------------------------------------

1. Introduction  
The Library Management System is a simple browser‑based tool that helps you manage books, members, borrowing, returning, due dates, and late fees. It runs directly in a web browser and does not require installation or a database.

------------------------------------------------------------
2. Starting the System  
1. Open the HTML file in any web browser (Chrome, Edge, Firefox, etc.).  
2. The system will load automatically and display the main dashboard.  
3. Demo data is already included so you can start using the system immediately.

------------------------------------------------------------
3. Adding a Book  
1. Go to the “Add / Edit Book” section.  
2. Enter the following details:  
   - Book ID  
   - Book Title  
   - Author  
   - Number of Copies  
3. Click the “Save Book” button.  
4. The book will appear in the Books table below.

------------------------------------------------------------
4. Editing a Book  
1. In the Books table, find the book you want to edit.  
2. Click the yellow Edit button next to it.  
3. The book details will load into the form.  
4. Make the necessary changes.  
5. Click “Save Book” to update the record.

------------------------------------------------------------
5. Deleting a Book  
1. In the Books table, click the red Delete button next to the book.  
2. Confirm the deletion.  
Note: You cannot delete a book that is currently borrowed.

------------------------------------------------------------
6. Adding a Member  
1. Go to the “Add / Edit Member” section.  
2. Enter the Member ID and Member Name.  
3. Click “Save Member”.  
4. The member will appear in the Members table.

------------------------------------------------------------
7. Editing a Member  
1. In the Members table, click the yellow Edit button next to the member.  
2. The details will load into the form.  
3. Make changes and click “Save Member”.

------------------------------------------------------------
8. Deleting a Member  
1. Click the red Delete button next to the member.  
2. Confirm the deletion.  
Note: You cannot delete a member who has borrowed books.

------------------------------------------------------------
9. Borrowing a Book  
1. Go to the “Borrow Book” section.  
2. Select a member from the dropdown list.  
3. Select a book from the dropdown list.  
4. Click the “Borrow” button.  
The system will:  
- Reduce the available copies  
- Add the book to the member’s borrowed list  
- Set a due date (7 days from today)  
- Add a borrow entry to the history log

------------------------------------------------------------
10. Returning a Book  
1. Go to the “Return Book” section.  
2. Select a member from the dropdown list.  
3. Select the book they borrowed.  
4. Click the “Return” button.  
The system will:  
- Increase available copies  
- Remove the book from the member’s borrowed list  
- Check if the book is overdue  
- Add late fees if necessary  
- Add a return entry to the history log

------------------------------------------------------------
11. Viewing Books  
1. Scroll to the Books section.  
2. You will see a table showing:  
   - Book ID  
   - Title  
   - Author  
   - Available copies  
   - Total copies  
   - Edit and Delete options

------------------------------------------------------------
12. Viewing Members  
1. Scroll to the Members section.  
2. You will see a table showing:  
   - Member ID  
   - Name  
   - Number of borrowed books  
   - Total late fees  
   - Edit and Delete options

------------------------------------------------------------
13. Viewing Borrow/Return History  
1. Scroll to the History section.  
2. You will see a list of all borrow and return events.  
Each entry shows:  
- Member name  
- Book title  
- Date and time  
- Late fee (if any)

------------------------------------------------------------
14. Late Fee Rules  
1. Each book has a due date of 7 days from the borrow date.  
2. If a book is returned after the due date, a late fee is charged.  
3. Late fee is calculated as:  
   1 currency unit per day late.  
4. Late fees are added to the member’s total balance.

------------------------------------------------------------
15. Demo Data  
The system includes sample Indian books and Indian member names so you can test the system immediately without entering data manually.

------------------------------------------------------------
16. Resetting the System  
Refreshing the page clears all data because the system uses in‑memory storage.  
(If you want data to be saved permanently, local storage can be added later.)

------------------------------------------------------------
17. Summary  
The Library Management System allows you to:  
- Add, edit, delete books  
- Add, edit, delete members  
- Borrow and return books  
- Track due dates and late fees  
- View complete history  
- Use a clean and professional interface
